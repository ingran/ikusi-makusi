CSS remove and combine: To remove the unsued selector from all stylesheets on a page 
and combines the result into a single stylesheet:

https://chrome.google.com/webstore/detail/css-remove-and-combine/cdfmaaeapjmacolkojefhfollmphonoh?hl=en-GB

Files:

 * bootstrap.min.css --> Bootstrap Style
 * bootstrap.min.beautified.css --> Bootstrap Style Beautified for better reading
 * custom.css --> Custom style mods for Bootstrap Style
 * combined.css --> Used an combined selectors of all the styles (bootstrap.min.css+custom.css) --> CSS remove and combine
 * ingran_style --> Reduced group of selector available in combined.css
